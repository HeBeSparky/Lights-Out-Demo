# Lights-Out-Demo
Lights Out! Demo

Keyboard controls:
Esc - start/pause
F - run
Space - select/jump
arrow keys - movement

Gamepad controls:
Start - start/pause
X - run
A - select/jump
joystick1 - movement 

After playing the game, please go to https://markandrewvb.typeform.com/to/MPuDdk and take the survey about the game.
